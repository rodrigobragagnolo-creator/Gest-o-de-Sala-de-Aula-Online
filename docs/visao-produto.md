# Definição do Problema

Atualmente, muitas instituições de ensino utilizam ferramentas separadas para gerenciar suas atividades acadêmicas, como aplicativos de mensagens, e-mails, planilhas e plataformas distintas para videoconferência.

Essa fragmentação gera diversos problemas, entre eles:

- Dificuldade na organização das turmas;
- Perda de informações importantes;
- Falta de centralização de atividades e avaliações;
- Comunicação ineficiente entre professores e alunos;
- Maior chance de erros administrativos;
- Retrabalho para professores e coordenadores.

Além disso, a ausência de uma plataforma unificada dificulta o acompanhamento do desempenho dos alunos, o controle de prazos de entrega e a análise de indicadores acadêmicos.

## Solução Proposta

O projeto propõe o desenvolvimento do **ClassFlow**, um sistema de gestão de sala de aula online inspirado em plataformas como Google Classroom.
