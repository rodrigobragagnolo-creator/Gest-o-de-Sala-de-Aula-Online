# Regras de Negócio

## RN01
Apenas professores podem criar atividades.

## RN02
Alunos só podem entregar atividades até a data limite.

## RN03
Apenas o professor da turma pode corrigir atividades.

## RN04
Coordenadores podem visualizar relatórios.

## RN05
Sistema envia notificações automáticas.

## RN06
Cada aluno possui apenas uma entrega por atividade.

## RN07
Usuários precisam estar autenticados.
