# Modelo C4

Esta pasta contém os diagramas C4 da arquitetura do sistema.

## Objetivo

Representar a arquitetura da solução em diferentes níveis de abstração.

## Diagramas

- `contexto.png` → Nível 1 (Contexto)
- `containers.png` → Nível 2 (Containers)

## Níveis

### Nível 1 — Contexto
Mostra os usuários e sistemas externos interagindo com o ClassFlow.

### Nível 2 — Containers
Mostra os principais componentes internos:

- Frontend Web
- Backend API
- Serviço de Notificações
- Banco de Dados
