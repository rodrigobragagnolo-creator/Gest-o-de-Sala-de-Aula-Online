# ClassFlow

Sistema de Gestão de Sala de Aula Online desenvolvido para a disciplina de **Modelagem e Projeto em Engenharia de Software**.

## Estrutura do Projeto

- `/docs` → documentação textual
- `/bpmn` → processos de negócio
- `/uml` → modelagem UML
- `/c4` → arquitetura C4
- `/apresentacao` → slides da banca

## Funcionalidade Inovadora

O sistema possui uma funcionalidade de **correção assistida por IA**, capaz de auxiliar professores na avaliação de atividades.
